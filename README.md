# Pastebin Lite

A simple Pastebin-like web application built as a take-home assignment.

## Features
- Create text pastes
- Generate shareable URLs
- View pastes using the link
- Backend health check endpoint

## Tech Stack
- Next.js (App Router)
- Node.js
- PostgreSQL (Neon)
- Deployed on Vercel

## Local Setup
1. Install dependencies
   npm install
2. Create .env.local
3. Run the app
   npm run dev

## Health Check
GET /api/healthz
